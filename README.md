# jobhunter
A simple job service front-end developed in ReactJs.

The project is divided in frontend and backend
 ./client/   (FrontEnd)
 ./api/      (BackEnd)

## Quick Start
To run the project you need two terminal windows:

Front end
```
cd client
npm start
```

Back End
```
cd api
npm start
```
## Requirements
 + ReactJs
 + Express.js
 + MongoDb
 + NodeJs v10+
